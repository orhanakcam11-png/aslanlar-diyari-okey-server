FEKE OKEY WEB SİTESİ

1. Zip dosyasını çıkar.
2. index.html dosyasını aç.
3. Netlify'a sürükle bırak yaparsan canlı link oluşur.

Bu sürüm statik demo web sitesidir.


GÜNCELLEME:
- Aslanlar Diyarı Okey görseli Feke Okey ana arayüzüne entegre edildi.
- Eğlence amaçlı uyarısı ana ekrana eklendi.


PROFESYONEL GÜNCELLEME:
- Premium tanıtım bandı eklendi.
- Özellik kartları eklendi.
- Cam/neon efektli arayüz güçlendirildi.
- Eğlence amaçlı uyarı daha belirgin hale getirildi.


GÜNCELLEME:
- Profesyonel giriş ekranı eklendi.
- Yeni Aslanlar Diyarı Okey logosu arayüze entegre edildi.
- Misafir girişi, giriş yap, kayıt ol ve Google ile giriş butonları eklendi.


GÜNCELLEME:
- Misafir girişi çalışır hale getirildi.
- Kayıt ol formu localStorage ile bağlandı.
- Giriş yap formu kaydedilen kullanıcıyı kontrol eder hale getirildi.
- Google ile giriş demo olarak bağlandı.
- Giriş sonrası lobiye yönlendirme eklendi.
